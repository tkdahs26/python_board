# python_board

파이썬 Flask프레임워크를 활용한 객체 지향 기반 간단 게시판 프로젝트<br>
자바에서 언어 범위를 넓히기위해 만드는 프로젝트.  __init__, self문 같은 파이썬 문법 구조를 이해하고 flask 프레임워크 문서를 파악할 정도의 실력 향상을 위해 만든 게시판입니다. flask 문서를 보며 파이썬을 이해했고 DB연동,자료형만 알면 자바와같은 채팅같은 프로그램도 만들 수 있겠다 생각했습니다
